      Help & privacy

      EULA / Terms

      `apply-bot` is a review-first helper. You control what gets filled and whether anything is submitted.

      Privacy

      Your profile, tracker, and memory stay in local Chrome storage by default. Optional resume parsing / AI calls only use the data needed for that request.

      What is an ATS?

      An applicant tracking system is the hiring software behind many job forms. `apply-bot` uses ATS detection to decide how safely it can fill fields and how much review to keep in the loop.

      GDPR / data control

      Use the actions below to clear temporary cache or fully delete your local extension data.
